How to Manage Series

Option 1: Main CLI Menu
  1. Open terminal/command prompt
  2. Navigate to your DieCast Tracker folder
  3. Run: python main.py
  4. Select option 9 for Series Management
  5. Follow the interactive menu
Option 2: Direct Access
  1. Open terminal/command prompt
  2. Navigate to your DieCast Tracker folder
  3. Run: python scripts/manage_series.py
  4. Use the interactive management interface
Available Management Options:
  • Add Main Series: Create new series categories
  • Add Subseries: Add new subseries to existing categories
  • Remove Subseries: Remove outdated subseries
  • Edit Metadata: Update descriptions, prices, and rarity
  • Export Configuration: Save changes to the system
